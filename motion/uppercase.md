# Uppercasing

## Python

```python
'yeet!'.upper() # YEET!
```

## JavaScript

```javascript
'yeet!'.toUpperCase(); // YEET!
```

## C#
 
```csharp
"yeet!".ToUpper(); // YEET!
```

## PHP

```php
strtoupper('yeet!');  # YEET!
```

## Java

```java
"yeet!".toUpperCase(); // YEET!
```

## Ruby

```ruby
'yeet!'.upcase # YEET!
```

## Low level bit code (Python)

```python
# Low level XOR of 0x20 will give you ASCII uppercase
chr(ord('y') ^ 0x20) # 'Y'
chr(ord('e') ^ 0x20) # 'E'
chr(ord('e') ^ 0x20) # 'E'
chr(ord('t') ^ 0x20) # 'T'
```
